Pico Car

Tools: Thonny and HTML | Source Code: GitHub

This is a Pico Car. The car can go forward, backward, turn right, and turn left. The car uses 2 DC Motors, Raspberry Pico W, batteries, and a powerbank. You can also find a folder named "Maze Solver" that has code for pico-car to solve a maze. This model uses an ultrasonic sensor in addition to the previous materials.

Here you can find the full project and its code etc.

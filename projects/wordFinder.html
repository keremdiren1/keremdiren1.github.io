Word Finder

Tools: Scratch | Source Code: GitHub

In this word finder, you can click word and then the + symbol to add words. The program will ask you to enter the word, it's translated version, and its language. You can then add a language filter by pressing the "Language Filter" button and then writing the language you want to add to the filter. The language can be added to the filter only after you write it into the language section. Writing "None" to the filter will reset it. You can also add a favorite filter by clicking the star button. It works the same as the language filter; the only difference is that you can only write words in here. After that, you can go back and then insert text after clicking the "insert text" button. You can see whether the word you're looking for is in there by clicking the "show / hide" button.

Taken Notes:

A simple code is usually started like this in java:

public class className{

- - - public static void main(String[] args){

- - - - - - /* some code */

- - - }

}

Writing the code System.out.print(text etc.) would print the text written in ().

System.out.println(text etc.) would also do the same; except, it would also go to the next line after the text is printed out.

Example:

System.out.print(“Hello ”);

System.out.println(“hi”);

System.out.print(“what”);

Would print out:

Hello hi

what

Syntax Error: Something is wrong with the code.

Logic Error: The output isn’t what we expect it to be.

Data types are types that represent some data.

Data Type Examples:

int data type represents integers. (12 or 3 etc.)

double data type represents numbers with decimal places. (1.0 or 15.23 etc.)

boolean data type represents the values true and false.

String data type represents text. (“Hello” or “12” or “oii” etc.)

Primitive data types are data types that have their values determined beforehand.

ints can have integers with 9 to 10 digits.

doubles can have numbers with 14 to 15 digits.

booleans can only have true or false values.

String is not a primitive data type because it does not have pre-determined values.

Variables are words that hold a space in the computer after they are created. We can set values to them if the type of data the variable has matches with the type of data we want to set. If we don’t set them a value, they would have a reserved empty space in the computer memory.

We can create variables like this:

(type of variable) (name of the variable);

int number;

We can also give an initial value to variables:

(type of variable) (name of the variable) = (value);

int number = 12;

Creating a variable has some rules:

Variables may consist of letters, digits, or an underscore. (case sensitive)

Variables may not start with a digit.

Spaces are not allowed in variables.

Variables may not use any other characters such as &, @, or $.

Variables ﻿may not use Java reserved words.

We can change the value of a variable:

int number = 3;

number = 18;

We can create variables that can’t have its value changed:

final int number = 15;

We can do math in java. Some basic symbols used are +, -, *, /, and %. % is used to get the remainder of a division.

Literal: The source code representation of a fixed value: 3

String Literal: Enclosed in double quotes: “3”

If we try to add (+) two strings, it will add them together.

Example:

System.out.println(“hel” + “lo”);

Will print out:

hello

Making an equation with doubles would give us a double result.

Making an equation with integers would give us an integer result.

Making an equation with doubles and integers would give us a double result.

Writing something like (variable += 12) would be the same as (variable = variable + 12). This also works for other symbols.

Writing (variable--) or (variable++) is the same as doing (variable –= 1) and (variable += 1)

Writing ((data type) variable) would turn the variable into the written data type.

Example:

double bb = 12.0;

int aa = (int) bb;

If the value of an int or a double is too much, the value would turn into the smallest value possible. If the value is too little, it will turn into the largest value possible.

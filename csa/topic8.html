AP CSA Topic 8

Taken Notes:

A 2D array is stored as an array of arrays. For example, a 2D array of ints would be an array of int arrays.

We can create 2D arrays like the following:


int[][] array = new int[row][col];
int[][] array = {{1, 2, 3}, {4, 5, 6}};

We can also create 2D arrays that are not rectangular (arrays like {{1, 2, 3}, {4, 5}}). This, however, is not a topic in the AP CSA exam.

In the exam, the first [] in “new int[row][col];” represents the rows, while the seconds [] represents the columns. An example 2 row and 3 column array would be “{{1, 2, 3}, {4, 5, 6}}”.

We can access a specific element by “array[rowindex][colindex]”, while we can access a specific row by “array[rowindex]”.

We can traverse through 2D arrays by using nested while loops, nested for loops, or nested enhanced for loops.

An example with enhanced for loops would be this:


for (int[] i : array){
	for (int k : i){
		System.out.println(k);
	}
}

The variable i selects a row, while the variable k traverses through the i each iteration.

We can get the number of rows by “array.length”, while we can get the number of columns by “array[0].length”.

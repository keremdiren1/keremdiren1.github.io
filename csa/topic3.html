-== checks whether the expressions in each of its sides are equal or not. If they are, it gives the boolean value true, if not, it gives false. (1 == 1) gives true while (2 == 1) does not. De Morgan’s Laws:

Writing ! in front of anything will reverse it. If there is an ! in front of something like (a == 0 && b == c), the whole expression would turn into (a != 0 || b != c).

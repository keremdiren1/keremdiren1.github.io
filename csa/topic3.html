-== checks whether the expressions in each of its sides are equal or not. If they are, it gives the boolean value true, if not, it gives false. (1 == 1) gives true while (2 == 1) does not. Other operators that do this are <, >, <=, >=, and !=. These mean smaller, greater, smaller than or equal to, greater than or equal to, or not equal, respectively (== means equal).

-An if statement is a statement that activates the code below it if the condition that is next to it is true.

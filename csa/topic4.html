AP CSA Topic 4

Taken Notes:

A while loop starts only when the condition of it is fulfilled. After it starts once, it doesn’t stop repeating until its condition is unfulfilled.

Example:


while (a<10){
	a++;
}
System.out.println(a);

This will print out 10 because a increases by 1 until the while loop stops, which happens when a is greater than or equal to 10.

If the condition is unfulfilled while the current loop isn’t finished, the loop continues until it gets to the end, then it finishes.

for loops are a bit more complicated than while loops. They also continue until their condition is fulfilled, but they can also have a private variable unique to them. You also need to initialize the variable, and you need to indicate how much it will increase every time it finishes.

Example:


for (int i = 0; i < 5; i++){
	System.out.print(i);
}

This would print out 01234 because the variable increases every time the loop finishes. The (int i = 0) part is the part that creates and initializes the private variable. The (i++) part is the part that shows how much the i will increase every time the loop finishes. The (i < 5) part is the condition of the loop.

The condition part doesn’t need to have the private variable (i in this case) in it, it can have any other variable.

Example:


int a = 0;
for (int i = 0; a < 13; i++){
	a += i;
}
System.out.println(a);

This would print out 15 because a is increased by i every time the loop starts and i is increased by 1 every time the loop ends. When the loop ends, which requires a to be more than or equal to 13, a is 15 because it was increased by i when i was 0, 1, 2, 3, 4, and 5.

A nested for loop is just a for loop inside a for loop, nothing much complicated.

Example:


for (int i = 0; i < 5; i++){
	for (int a = 0; a < i; a++){
		System.out.print("*");
	}
	System.out.println();
}

This would print:



*
**
***
****

Or you can do this:


for (int i = 0; i < 5; i++){
	for (int a = 0; a < 3; a++){   // i is changed to 3
		System.out.print(“*”);
	}
	System.out.println();
}

This would print:


***
